# 移动端首页布局实现计划

## 需求概述
构建 Vue 3 + Vant 移动端首页布局：
- 底部导航栏：首页、工作台、我的
- 首页内容：数据展示（统计卡片 + 图表 + 消息列表）
- 运行环境：企业微信 H5

## 企业微信适配说明
企业微信 H5 应用无法完全隐藏自带的顶部导航栏，因此：
- **暂不添加顶部导航栏**，避免与企业微信导航重复
- 后续如需自定义导航，可通过企业微信 JSSDK 配置

## 技术要点

1. **Vant Tabbar**：使用 `van-tabbar` + `van-tabbar-item` 实现底部导航
2. **路由懒加载**：页面组件使用 `() => import(...)` 按需加载
3. **路由同步**：监听路由变化，同步更新 tabbar 激活状态
4. **移动端适配**：viewport 设置禁止缩放，CSS 使用 px 单位
