# 工作台页面开发计划

## 需求概述
开发工作台页面，包含四个功能菜单入口：
- 单据审批
- 工时反馈
- 冲焊检具
- 质量快反

## 布局方案
采用 **卡片式布局**，每个菜单项都是一个独立的卡片，卡片之间有 12px 间距。

```
┌─────────────────────────────┐
│  [图标]  单据审批    >      │  ← 独立卡片
└─────────────────────────────┘
        ↕ 12px 间距
┌─────────────────────────────┐
│  [图标]  工时反馈    >      │  ← 独立卡片
└─────────────────────────────┘
        ↕ 12px 间距
┌─────────────────────────────┐
│  [图标]  冲焊检具    >      │  ← 独立卡片
└─────────────────────────────┘
        ↕ 12px 间距
┌─────────────────────────────┐
│  [图标]  质量快反    >      │  ← 独立卡片
└─────────────────────────────┘
```

## 样式设计
- 使用 `menu-list` 容器 + `van-cell-group inset` 实现独立卡片
- 卡片间距：`gap: 12px`（与首页风格一致）
- 白色卡片背景，圆角 8px
- 左侧图标 + 标题 + 描述 + 右侧箭头
- 页面灰色背景 #f5f5f5

## 工时反馈子菜单功能 ✅

### 需求
- 工时反馈菜单点击后不跳转页面，而是弹出子菜单
- 子菜单包含三个选项：钳调反馈、其他工时、复核工时
- 采用并排 icon 布局，每个选项有独特图标和颜色
- 子菜单点击暂不实现跳转（后续开发）

### 交互流程
```
点击"工时反馈"
       ↓
┌─────────────────────────────┐
│       选择工时类型           │
│                             │
│   [编辑]   [时钟]   [清单]   │
│   钳调反馈  其他工时  复核工时 │
│                             │
│  ─────────────────────────  │
│           取消              │
└─────────────────────────────┘
       ↓
点击选项 → 显示 Toast 提示（后续实现跳转）
```

### 图标配色
| 选项 | 图标 | 颜色 |
|------|------|------|
| 钳调反馈 | edit | #1989fa (蓝) |
| 其他工时 | clock-o | #07c160 (绿) |
| 复核工时 | todo-list-o | #ff976a (橙) |

### 技术实现
- 使用 `van-popup` + `van-grid` 组件实现并排 icon 布局
- 通过 `showWorkhourSheet` ref 控制弹出状态
- `workhourActions` 定义子菜单选项（含 icon、color 属性）
- `onWorkhourSelect` 处理选择事件，先关闭弹窗再执行操作
- 底部取消按钮点击关闭弹窗
